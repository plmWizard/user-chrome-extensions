Fusion Manage Extension Settings

Flip features on or off and optionally swap the Fusion logo with your own.

Second Tabs

Shows the extra tab row under native Fusion Manage tabs and loads tab URLs in an iframe.

Search

Loads the Fusion Manage view starting with the name Search and shows a live-searchable overlay grid on workspace list pages.

Action Buttons

Shows script-based action buttons on tabs when displayName uses mode: button.

On-Edit Runner

Runs scripts tagged with mode: onEdit when edit changes back to view or after BOM/attachment writes.

Workspace Header Colors

Reads {color: #RRGGBB} from workspace description and applies theme variables.

Item Context Badge

Shows a compact workspace/item badge with quick copy actions on item pages.

Scroll To Top Button

Adds a floating button to quickly return to the top on long pages.

DMS Bulk Copy/Add Tools

On /items list pages, lets you select and copy multiple dmsIDs. On BOM and Affected Items tabs, paste/load copied dmsIDs and add them in bulk.

Custom Logo Image URL

If this field is empty, the native Fusion logo is left untouched.

Custom Logo Click URL

Optional. If filled, clicking the custom logo goes here instead of mainDashboard.

After changing settings, refresh Fusion Manage if the page does not update instantly.
